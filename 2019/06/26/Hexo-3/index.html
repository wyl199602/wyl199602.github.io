Markdown

1、Markdown 简介

Markdown 是一种可以使用普通文本编辑器编写的 标记语言，通过简单的 标记语法，它可以使普通文本内容具有一定的格式。

2、编辑器

关于Markdown，有一个官方的编辑器——Typora
按照官方的说法就是 简单而强大，它不仅支持原生的语法，也支持对应的快捷键，更重要的是它还可以 实时预览

个人用的是VsCode，鉴于VsCode强大的扩展器。
而且VsCode安装了插件之后也可以实时预览。
